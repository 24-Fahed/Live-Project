# HBuilderX 运行模式说明

## 1. 本地模拟器

- 地址：`http://localhost:8080`
- 场景：本地 Docker 已启动，使用微信开发者工具模拟器联调

## 2. 真机局域网

- 运行项：`mp-weixin-device`
- 地址：`http://192.168.137.1:8080`
- 场景：手机和开发机处于同一局域网

## 3. IP 调试

- 运行项：`mp-weixin-ip-debug`
- 地址：`http://106.14.254.19:8080`
- 场景：后端已部署到公网服务器，需要真机联调

## 4. IP 上线

- 运行项：`mp-weixin-ip-release`
- 地址：`http://106.14.254.19:8080`
- 场景：小程序使用公网 IP 对接正式服务
